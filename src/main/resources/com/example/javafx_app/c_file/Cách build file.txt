Nếu éo tải visual studio 2022
Bước 1: Vào visual studio 2022 (Éo có thì tải đi) Nó sẽ tải cả visual studio 2022 cả visual studio installer
Bước 2: Vào visual studio installer, bấm vào Modify, tìm mục Desktop & Mobile, chọn Desktop development with C++, sau đó bấm Modify(Hoặc Install)
Bước 3: Tạo file C++ mới, đặt cho repos một tên đi
Bước 4: Copy code trong file cpp vào, sau đó chỉnh sửa tùy ae:)
Bước 5: Click chuột phải vào tên project, chon Properties -> Configuration Properties -> Linker -> System
Bước 6: Thay SubSystem từ Console (/SUBSYSTEM:CONSOLE) thành Windows (/SUBSYSTEM:WINDOWS)
Bước 7: Nhấn tổ hợp Ctrl + Shift + B để build
Bước 8: Tìm trên máy đường dẫn source\repos\<Tên repos>\x64\Release rồi kéo thả (Nhớ giữ phím Ctrl) file exe vào file c_file
À mà nhớ đưa mấy code c++ luôn vào file c_file (Tạo file mới + <Tên_file>.cpp rồi copy code) cho ae xem nha:)
Ước gi IntelliJ nó build đc file C++ thì éo lòng vòng như thế này
